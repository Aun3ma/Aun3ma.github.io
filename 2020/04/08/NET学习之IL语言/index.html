.NET学习之IL语言

.NET IL语言
.NET IL语言
发布日期:   2020-04-08
阅读次数:  

IL语言（也称为公共中间语言CIL,或微软中间语言MSIL）是.NET的核心，以.NET Framework平台运行的程序最终都是由IL代码实现。
也可以这样理解，无论C/C++、BASIC、Pascal等语言特征如何，在INTEL芯片上最终执行的都是Intel汇编指令集。
了解IL相关知识能够以较高层次理解.NET工作原理，最大程度利用.NET Framework进行高级程序开发。

学习IL语言首先需要学会使用ildasm反汇编工具，通过ildasm工具，我们可以将任意的.NET的dll或exe文件反汇编为MSIL代码。

为了更好的解释IL语言，这里我编写了一个十分简单的C#程序，仅用于学习和了解IL语言。

构造函数ctor

构造函数.ctor中的内容如图四所示：

其中，.ctor表示构造函数。
cil managed表示方法体重的代码是IL代码，并且是托管代码，运行在CLR运行库中。
.maxstack表示执行构造函数时，Evaluation Stack可容纳数据项的最大个数。Evaluation Stack是保存方法中所需变量值的一块内存区域，在方法执行结束时被清空或存储一个返回值。这里Evaluation Stack为8。
IL_0000为代码行开头，ldarg.0指令意为加载第一个成员参数。
IL_0001：Call instance void [mscorlib]System.Object::.ctor()意为调用System.Object类的构造函数。通常Call指令一般用于调用静态函数。
IL_0006：nop表示不执行任何操作。
IL_0007：ret指令表示执行完毕返回。

下面解释IL_标记之后的部分，内容如图六和图七所示：

IL_0000：nop指令表示不执行任何操作。
IL_0001：ldstr bytearray (F7 8B 93 8F 65 51 60 4F 84 76 0D 54 57 5B 21 00 )指令表示将bytearray中的内容的Reference放进Evaluation Stack。bytearray中的内容即“请输入你的名字！”的十六进制编码。
IL_0006：Call void [mscorlib] System.Console::WriteLine (string)指令表示调用System.Console::WriteLine函数并取出Evaluation Stack中的第一个值作为参数。即输出“请输入你的名字！”。
IL_000b：nop指令表示不执行任何操作。
IL_000c：Call string [mscorlib]System.Console::ReadLine()指令表示调用System.Console::ReadLine()函数。该函数将获得的值放入Evaluation Stack。
IL_0011：stloc.0指令将Evaluation Stack中刚刚通过System.Console::ReadLine()函数读入的字符串放入变量V0中。
IL_0012：ldstr bytearray (22 6B CE 8F 60 4F 21 00 7B 00 30 00 7D 00 ) 指令表示将bytearray中的内容的Reference放进Evaluation Stack。bytearray中的内容即“欢迎你!”的十六进制编码。
IL_0017：ldloc.0指令将V0变量的值放入Evaluation Stack。
IL_0018：call void [mscorlib] System.Console::WriteLine (string,object)指令调用System.Console::WriteLine函数并取Evaluation Stack中的前两个值作为参数。即输出”欢迎你!xxx”。
IL_001d：nop指令表示不执行任何操作。
IL_001e：ldstr “Hello” 指令表示将字符串“Hello”的Reference放进Evaluation Stack。
IL_0023：stloc.1指令将Evaluation Stack中刚刚放入的字符串“Hello”放入变量V1中。
IL_0024：ldstr “ ”指令表示将字符串“ ”的Reference放进Evaluation Stack中。
IL_0029：stloc.2指令将Evaluation Stack中刚刚放入的字符串“ ”放入变量V2中。
IL_002a：ldloc.1指令将V1中的值放入Evaluation Stack。
IL_002b：ldloc.2指令将V2中的值放入Evaluation Stack。
IL_002c：ldstr “World”指令表示将字符串“World”的Reference放进Evaluation Stack中。
IL_0031：call string [mscorlib] System.String::Concat (string,string,string)指令调用System.String::Concat函数并取Evaluation Stack的前三个值作为参数。即对三个字符串做拼接。
IL_0036：stloc.3指令将刚刚拼接好的字符串存入变量V3中。
IL_0037：ldloc.3指令将V3中的值放入Evaluation Stack。
IL_0038：call void [mscorlib] System.Console::WriteLine (string)指令调用System.Console::WriteLine函数并取Evaluation Stack中的第一个值作为参数。即输出“Hello World！”。
IL_003d：nop指令表示不进行任何操作。
IL_003e：ldstr bytearray (F7 8B 93 8F 65 51 30 00 16 62 31 00 01 FF ) 指令表示将bytearray中的内容的Reference放进Evaluation Stack。bytearray中的内容即“请输入0或1！”的十六进制编码。
IL_0043：call void [mscorlib] System.Console::WriteLine (string)指令调用System.Console::WriteLine函数，并取Evaluation Stack中的第一个值作为参数。即输出“请输入0或1！”。
IL_0048：nop指令表示不进行任何操作。
IL_0049：call string [mscorlib]System.Console::ReadLine()指令调用System.Console::ReadLine函数。
IL_004e：stloc.s V_4指令将读到的值放入V4变量中。
IL_0050：ldloc.s V_4指令将V4的值放入Evaluation Stack。
IL_0052：call int32 [mscorlib]System.Int32::Parse(string)指令调用System.Int32::Parse函数，并使用Evaluation Stack中的第一个值作为参数。即将输入的字符串转换为int类型。
IL_0057：stloc.s V_5指令将转换过后的值存入V5变量。
IL_0059：ldloc.s V_5指令将V5中的值放入Evaluation Stack。
IL_005b：ldc.i4.1指令表示在Evaluation Stack中放入一个4 byte的常数，其值为1。
IL_005c：ceq指令比较Evaluation Stack中的前两个值。如果这两个值相等，则将整数值1 (int32)推送到Evaluation Stack上；否则，将0 (int32)推送到Evaluation Stack上。
IL_005e：stloc.s V_6指令将Evaluation Stack中的第一个值取出，存入V6变量。
IL_0060：ldloc.s V_6指令将V6中的值放入Evaluation Stack。
IL_0062：brfalse.s IL_0073指令表示如果value为false、空引用或零，则将控制转移到IL_0073。
IL_0064：nop指令表示不进行任何操作。
IL_0065：ldstr “True！”指令将字符串“True！”的Reference放入Evaluation Stack中。
IL_006a：call void [mscorlib]System.Console::WriteLine (string)指令调用System.Console::WriteLine函数，并取Evaluation Stack中的第一个值作为参数。即输出“True！”。
IL_006f：nop指令表示不执行任何操作。
IL_0070：nop指令表示不执行任何操作。
IL_0071：br.s IL_0080指令表示无条件转移到IL_0080指令。
IL_0073：nop指令表示不执行任何操作。
IL_0074：ldstr “False！”指令将字符砖“False！”的Reference放入Evaluation Stack。
IL_0079：call void [mscorlib]System.Console::WriteLine (string)指令调用System.Console::WriteLine函数，并取Evaluation Stack中的第一个值作为参数。即输出“False！”。
IL_007e：nop指令表示不执行任何操作。
IL_007f：nop指令表示不执行任何操作。
IL_0080：ret指令表示执行结束返回。由于Main函数是Void类型，所以没有返回值且Evaluation Stack为空，程序正常结束。
